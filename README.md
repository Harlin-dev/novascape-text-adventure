# Novascape Text Adventure Game

A Python text-based adventure game created for my **Introduction to Scripting** college course.  

Explore a space station, collect essential items, and escape while avoiding the villain, Nova.  

**Tech:** Python | Console Game | Loops & Dictionaries | Functions

In **Novascape**, you play as a space explorer trapped on a space station. Your goal is to:

The game is entirely **text-based** and runs in the console.
